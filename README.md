🔐 Private Decryptor Tool

A simple, fast, and mobile-first web tool to decrypt encrypted messages instantly.

This tool is designed for personal use, allowing secure message reading without requiring manual key input.

---

🚀 Live Usage

1. Open the decryptor website
2. Paste the encrypted message
3. Message is automatically decrypted

No buttons. No extra steps. Clean and instant.

---

⚙️ Features

- 🔓 Automatic decryption (no button required)
- 📱 Mobile-first responsive design
- ⚡ Fast and lightweight (no backend)
- 🎯 Simple and focused UI
- 📋 Copy decrypted message with one tap
- 🧹 Clear input/output instantly

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- CryptoJS (AES encryption)

---

📁 Project Structure

/
 ├── index.html   # Main decryptor file

---

🔐 How It Works

- Messages are encrypted using AES encryption
- A predefined internal key is used for decryption
- The tool automatically detects and decrypts valid messages

---

⚠️ Important Note (Read This)

This project is built for personal/private use only.

It is NOT fully secure because:

- The logic runs in the browser
- The encryption key is embedded in the code
- Anyone with technical knowledge can inspect and reverse it

Do NOT use this for:

- Sensitive data
- Financial information
- Production-level security needs
